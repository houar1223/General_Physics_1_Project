# General Physics Project

## Project Overview

This project is about measuring gravitational acceleration using Bernoulli's equation.
The experiment analyzes the motion of water flowing out of a tank and compares the experimental results with the theoretical model.

This project includes experimental data, analysis code, simulation code, and result graphs related to the measurement of gravitational acceleration.
